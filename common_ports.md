# Common Ports

## Ports

22 - - SSH

53 - TCP/UDP - DNS

80 - TCP - HTTP

88 - TCP/UDP - Kerberos

135 - TCP/UDP - Remote Procedure Call (RPC)

139 - TCP/UDP - NetBIOS or Server Message Block (SMB)

389 - - LDAP

445 - TCP - Server Message Block (SMB)

636 - - LDAP over SSL (LDAPS)

3389 - - RDP Remote Desktop Protocal
